
# Petition App Client
SENG365 2020 Assignment 1 [A+]

## Technologies used
- Vue.js
- Bulma
- Vue-axios
- Vue-router
- Axios

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
